# Wine Not - Backend

## Descripción

API REST desarrollada con FastAPI para un e-commerce de vinos.

## Tecnologías

- Python
- FastAPI
- SQLAlchemy
- SQLite
- Pytest

## Estructura

app/
tests/

## Instalación

python -m venv venv

pip install -r requirements.txt

uvicorn app.main:app --reload

## Documentación

Swagger:

http://localhost:8000/docs

ReDoc:

http://localhost:8000/redoc

## Endpoints

- Wines
- Categories
- Feedback
